== Description ==

Plugin de integração da plataforma [Balcão Balcão](http://balcaobalcao.com.br/) para Wordpress.

== Frequently Asked Questions ==
= Como encontrar meu token de integração =
* Você deve efetuar o cadastro na Balcão Balcão como Pessoa Jurídica
* Migrar sua conta para e-commerce.
* Depois acesse Configurações > Token Api

== Installation ==
= Minimum Requirements =

* Recomenda-se PHP 7.2 ou maior
* Recomenda-se MySQL 5.6 ou maior

Após instalar e ativar o plugin, acesse o seu painel administrativo, WooCommerce, Configurações, Entrega, Balcão Balcão. Configure os campos e clique em Salvar alterações.

== Upgrade Notice ==
* Necessário possuir o plugin [Brazilian Marke on WooCommerce](https://wordpress.org/plugins/woocommerce-extra-checkout-fields-for-brazil/)
* Configurações de links permanentes devem estar ativas e diferentes do padrão, com a configuração padrão não irá funcionar os callbacks.
